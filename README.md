# Opf.NET

Opf.NET is a pure, modern C# port of the [OpenAI Privacy Filter](https://github.com/openai/privacy-filter). It provides a bidirectional token-classification model for personally identifiable information (PII) detection and masking in text, without requiring Python or PyTorch.

## Features
- Fast, local, on-premises PII detection.
- Pure C# implementation utilizing SIMD/Tensors for performance.
- Downloads HuggingFace `.safetensors` checkpoints directly.
- Handles custom Viterbi CRF sequence decoding.
